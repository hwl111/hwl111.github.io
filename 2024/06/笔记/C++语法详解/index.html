11. 其他常用语法与技巧

  • 命名空间：namespace myns { ... }
  • auto 关键字（类型自动推断，C++11）：auto x = 10;
  • range-based for（C++11）：for (auto x : v) { ... }
  • lambda 表达式（C++11）：auto f = [](int x){ return x*x; };
  • constexpr 常量表达式（C++11）：constexpr int sq(int x) { return x*x; }
  • 枚举类型：enum Color { RED, GREEN, BLUE };
